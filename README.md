üìñ Description
---------------
The Color Recall game is a static website project built using HTML, CSS, and JavaScript. The game is designed to test the player's memory and reaction skills by displaying a sequence of colored buttons and asking the player to recall the sequence. The game starts with a simple sequence and gradually increases the difficulty level as the player progresses.

The game is implemented using a combination of HTML for structure, CSS for styling, and JavaScript for logic. The game sequence is generated randomly, and the player's input is compared with the original sequence to determine the accuracy. The game also keeps track of the player's score and level, providing a challenging and engaging experience.

The Color Recall game is a fun and interactive way to improve cognitive skills, particularly memory and attention. The game can be played by people of all ages, making it a great tool for families, schools, and cognitive training programs. With its simple yet addictive gameplay, the Color Recall game is sure to provide hours of entertainment and mental stimulation.
